# Momentum-Aware ML Models for Predicting NBA Shot Accuracy

This repo includes the code for my Spring 2026 Independent Work Project.

There are two notebooks in this repo:
* `NBA_Momentum_EDA_Cleaning.ipynb`: Data merging, cleaning, and momentum feature engineering
* `IW_Model_Creation.ipynb`: Model training, evaluation, and results

## Data

Use the links below to access the two datasets used in this project.

**NBA Shot Location Data (2004–2025):** https://github.com/DomSamangy/NBA_Shots_04_25

**NBA Play-by-Play Data (1997–2023):** https://www.kaggle.com/datasets/szymonjwiak/nba-play-by-play-data-1997-2023

Create a folder called `data/` in the main directory and add the dataset files there. The notebooks automatically look for them!

This project only uses data from the **2021, 2022, and 2023 seasons** from both datasets.
